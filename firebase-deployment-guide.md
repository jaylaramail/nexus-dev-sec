### üöÄ Deployment Steps:

1. **Build the project**:
   ```bash
   npm run build
   ```

2. **Install Firebase CLI**:
   ```bash
   npm install -g firebase-tools
   ```

3. **Login to Firebase**:
   ```bash
   firebase login
   ```

4. **Initialize Firebase project**:
   ```bash
   firebase init hosting
   ```

5. **Deploy**:
   ```bash
   firebase deploy
   ```
